1. 자동완성 끄기
프로그램 실행 후 설정(Ctrl + , 또는 Command + ,)으로 들어간 후 검색항목에 intelliSense를 검색하면 
여러 체크박스 리스트를 볼 수 있습니다.
위(Show Classes)에서부터 아래(Show Words)까지 전부 체크 해제를 하면 자동완성 기능이 비활성화 됩니다.

2. 마우스로 이동시 마지막에 한글 중복 출력되는 오류
컴퓨터의 설정 > 언어 검색 > 시간 및 언어 > 언어 및 지역 > 한국어 : 언어옵션 > 
Microsoft 입력기 : 키보드 옵션 > 호환성 (Microsoft IME) 켜기

3. 실행 명령어 설정하기
VS 설정에서 바로가기 키 > run python file 검색 > 단축키 설정 ( ctrl + shift + enter )
